# Desenvolvimento

## Descrição
Para desenvolvedores que venham a dar suporte a esta aplicação.

Repositório inicial do TypeScript do framework [Nest](https://github.com/nestjs/nest).

## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Teste

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
